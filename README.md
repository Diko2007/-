# Бинарное Дерево Поиска (Binary Search Tree) на Python

Этот проект представляет собой простую реализацию **бинарного дерева поиска (BST)** с функцией визуализации структуры в консоли.

## 📖 Теория

**Бинарное дерево поиска** — это иерархическая структура данных, обладающая следующими свойствами:
* У каждого узла есть не более двух потомков (левый и правый).
* **Левый потомок** содержит значение, которое *меньше* значения родителя.
* **Правый потомок** содержит значение, которое *больше* значения родителя.

### Сложность алгоритмов:
* **Поиск:** $O(\log n)$ в среднем.
* **Вставка:** $O(\log n)$ в среднем.
* **Пространственная сложность:** $O(n)$ для хранения $n$ узлов.

---

## 🛠 Функционал

В данном проекте реализованы:
1. **Класс Node**: Создает узел дерева с полями `left`, `right` и `val`.
2. **Функция insert**: Рекурсивно добавляет новые значения в дерево, сохраняя его структуру.
3. **Функция display_tree_advanced**: Отрисовывает дерево в терминале с использованием псевдографики, наглядно показывая связи «Родитель — Потомок».
